# Servicio de Gestión de Clientes - Quarkus

Este proyecto es un servicio REST desarrollado con Quarkus que permite gestionar clientes con operaciones CRUD (Crear, Leer, Actualizar, Eliminar).

## Características

- API RESTful para gestión de clientes
- Base de datos H2 en memoria
- Implementación con Quarkus y Hibernate ORM Panache
- Soporte para ejecución en contenedores Docker

## Requisitos

- Java 17
- Maven 3.8+
- Docker (opcional, para ejecución en contenedores)
